用語読み意味
奇門遁甲きもんとんこう時間と方位から物事の吉凶を読む東洋占術。三式の一つ。
三式さんしき太乙・奇門・六壬。術数の最高峰とされる三つ。
九宮 / 洛書きゅうきゅう / らくしょ盤の舞台となる3×3のマスと、その並びの根拠の魔方陣。
三才さんさい天・人・地。盤の層構成の基礎。
地盤 / 天盤 / 人盤 / 神盤ちばん 他盤の四層。地＝下地、天＝九星と天盤干、人＝八門、神＝八神。
八門はちもん休生傷杜景死驚開。人の行動の吉凶を司る。
九星きゅうせい天蓬〜天禽の九つ。気の質・才能。
八神はっしん値符・六合・白虎ほか。無形の作用・場の空気。
三奇さんき乙・丙・丁。盤を吉に傾ける貴い干。
六儀ろくぎ戊己庚辛壬癸。甲が遁れ隠れる器。
遁甲とんこう最も尊い甲が六儀の陰に隠れること。術名の由来。
値符 / 値使ちふ / ちし盤の司令塔。九星の長と八門の長。
旬首じゅんしゅその旬を率いる六儀。値符・値使を決める起点。
局 / 陽遁 / 陰遁きょく 他18種の盤の型。冬至から陽遁、夏至から陰遁。
三元 / 符頭さんげん / ふとう一節気を上中下に三分する区切りと、その目印（甲・己の日）。
用神ようじん占う事柄・人物を盤上で代表させる記号。
落宮らっきゅう用神が落ちた宮。その状態を読む。
旺相休囚死おうそうきゅうしゅうし季節による五行の強弱の五段階。
格局かっきょく干門星神の組合せが作る吉凶の型。
空亡くうぼう力が抜ける宮。吉は半減、凶は緩む。
門迫もんぱく門が宮に剋され、力を発揮できない状態。
反吟 / 伏吟はんぎん / ふくぎん対冲の動揺（反吟）／伏す停滞（伏吟）。ともに凶。
入墓にゅうぼ干が墓の宮に入り力を失うこと。
暗干あんかん八門が裏に隠す第三の干。内実・裏事情を読む。
立向盤 / 座山盤りっこうばん / ざざんばん方位術の二盤。動いて出る気／居て受ける気。
転盤式 / 飛宮式てんばんしき / ひきゅうしき盤の組み方の二系統。回す／飛ばす。
趨吉避凶すうきつひきょう吉に向かい凶を避ける、奇門の実践理念。
応期おうき物事が起きる時期。盤から推す。
